# L2 Hamburgo Client

Public distribution repository for the L2 Hamburgo Interlude launcher and client
packages.

## Download

Open [Releases](https://github.com/joaquinzapata1/l2-client-updater/releases),
select the latest version and download only `InterludeLauncher.exe`.

The launcher asks where to install the game, then downloads, verifies and installs
the complete client. It also supports incremental updates, repair, cancellation,
backup and rollback.

The executable is not digitally signed yet, so Windows may display an unknown
publisher warning during this testing phase.

Source files and build scripts are maintained separately in the main project.
